# POETools

## Preparing the environment
Create a virtual environment with python:
> python -m venv venv
Activate it
- Windows:
> .\venv\Scritps\activate 
- Linux:
> source ./venv/bin/activate

Install required libs:
> pip install -r requirements.txt


## Executing scripts
> python \<**script name**\> \<**script argumnets**\>
